Debug

開発者用のコマンドです。全てのコマンドで開発者権限が必要です。

create

ファイルを生成します。

restart

BOTを再起動します。
その際、各変数はファイルに保存されます。

exit

BOTを停止します。
その際、各変数はファイルに保存されます。

py

任意のpythonコードを実行します。
jishakuにも同様の機能があります。(jsk py [code])

sh

任意のシェルスクリプトを実行します。
jishakuにも同様の機能があります。(jsk sh [code])

save

各変数をファイルに保存します。

cog

cogのロード/アンロード/リロードが行えます。
