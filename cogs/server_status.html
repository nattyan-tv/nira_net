Server status

Steam個人サーバーのステータスを表示します。
クールダウンは10秒/サーバーです。

ss add [表示名] [アドレス],[ポート番号]

指定したアドレス及びポート番号をその表示名で設定します。

[表示名]

何でも構いません。わかりやすいのがいいですね。

[アドレス]

ないと思いますがアドレスにはコンマ(カンマ)「,」を入れないでください。

[ポート番号]

Query Portと呼ばれる方を入れてください。

ss list

サーバーのリストを送信します。

ss auto [on/off]

サーバーがダウンしている際に通知を行います。30分ごとにチェックします。
鯖落ちの判定が緩いのでもしかしたらすぐ通知されるかもです。

[on/off]

有効にするか無効にするかです。

ss edit [サーバーナンバー] [表示名] [アドレス],[ポート番号]

指定したサーバーナンバーのサーバー情報を編集します。

[サーバーナンバー]

「ss list」を実行するとサーバーナンバーが表示されます。

[表示名]

何でも構いません。わかりやすいのがいいですね。

[アドレス]

ないと思いますがアドレスにはコンマ(カンマ)「,」を入れないでください。

[ポート番号]

Query Portと呼ばれる方を入れてください。

ss del [all/サーバーナンバー]

指定したサーバーナンバーのデータを削除します。
allと指定するとサーバーの情報すべてを削除します。

[all/サーバーナンバー]

サーバーナンバーは「ss list」を実行すると表示されます。
allにすると全てのデータが削除されます。

ss

リストに登録されたサーバーの情報を表示します。

ss [all/サーバーナンバー]

指定したサーバーナンバーの情報を表示します。
allと指定すると全てのサーバーの情報を詳細に表示します。

[all/サーバーナンバー]

サーバーナンバーは「ss list」を実行すると表示されます。
allにすると全てのサーバーの情報が詳細に表示されます。
